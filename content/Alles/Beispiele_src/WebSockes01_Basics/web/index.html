WebSockets-Beispiel 01

Dieses Beispiel illustriert die grundlegende Herangehensweise beim Einsatz von WebSockets. Das notwendige Javascript ist in eine separate Datei ausgelagert.

Der WebSockets-Server

NetBeans stellt für die Erstellung von WebSockets-Servern eine entsprechende Vorlage bereit: Sie ist in der Kategorie Web erreichbar.

Der zweite Schritt des Dialogs führt durch die notwendigen Angaben für die Erstellung eines Websocket-Endpunkt. Neben den Standard-Eigenschaften für die Endpunkt-Klasse (der Endpunkt ist ein POJO!) ist zunächst lediglich die Angabe des WebSockets-Identifyers erforderlich.

Quellcode

Der Quellcode eines minimalistischen Socket-Servers ist überschaubar:

@ServerEndpoint("/endpoint")
public class MeinWSEndpoint {

  @OnMessage
  public String onMessage(String message) {
    return "Hier folgt die Nachricht";
  }

}

“Und in echt…?”

Das minimalistische Beispiel tut nicht viel Sinnvolles, es verdeutlicht zumindest jedoch die Bandbreite möglicher Einsatz-Szenarien: WebSocket-Kommunikation ist überall dort einsetzbar, wo der Inhalt einer Nachricht in Text-Form übertragen werden kann.
