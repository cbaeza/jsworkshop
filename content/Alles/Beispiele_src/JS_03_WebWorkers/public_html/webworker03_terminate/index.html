Einen Worker beenden

Mit der terminate()-Methode des Workers können wir einen Worker beenden.

Bitte Schalter drücken ...

Quelltext

In der Seite: 

      var worker = new Worker('mein_worker.js');
      worker.onmessage = function(event) {
        console.log("Nachricht vom Worker : " + event.data);
      };

      worker.postMessage('Hallo');

      function beendeWorker(){
        console.log('Beende Worker.');
        worker.terminate();
      }
